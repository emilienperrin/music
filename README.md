# music  
## user guide:  
### server side:  
1. install the required packages (conda env create -f environment.yml)  
2. go there: cd server/
3. setup HTTPS certificates (see httpsguide.txt)
4. ./START_server.sh  (chmod +x before if needed)
  
### client side:  
1. open https://<ipServer>:8000 in a web browser
